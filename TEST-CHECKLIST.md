# Test-Checkliste

- Plugin startet ohne Fehler.
- Sicherheits-/Moderationsfunktionen greifen bei den vorgesehenen Aktionen.
- Chat- oder Schutzregeln blockieren nur das Richtige.
- Berechtigte Spieler/Admins koennen Ausnahmen weiterhin nutzen.
- Keine False Positives oder Konsolenfehler bei normalem Spielbetrieb.
